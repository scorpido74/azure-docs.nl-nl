# <a name="storage-explorer-accessibility"></a>Storage Explorer toegankelijkheid

## <a name="screen-readers"></a>Scherm lezers

Storage Explorer ondersteunt het gebruik van een scherm lezer in Windows en Mac. De volgende scherm lezers worden aanbevolen voor elk platform:

Platform | Scherm lezer
---------|--------------
Windows  | NVDA
Mac      | Voice-over
Linux    | (scherm lezers worden niet ondersteund in Linux)

Als u een toegankelijkheids probleem ondervindt bij het gebruik van Storage Explorer, [opent u een probleem op github](https://github.com/Microsoft/AzureStorageExplorer/issues).

## <a name="zoom"></a>Zoom

U kunt de tekst in Storage Explorer groter maken via inzoomen. Als u wilt inzoomen, klikt u in het menu Help op **Inzoomen** . U kunt ook het menu Help gebruiken om uit te zoomen en het zoom niveau weer in te stellen op het standaard niveau.

![Zoom opties in het Help-menu][0]

De zoom instelling neemt de grootte van de meeste UI-elementen toe. Het is raadzaam om ook grote tekst-en zoom instellingen in te scha kelen voor uw besturings systeem om ervoor te zorgen dat alle elementen van de gebruikers interface op de juiste wijze worden geschaald.

## <a name="high-contrast-themes"></a>hoog contrast-Thema's

Storage Explorer heeft twee Thema's met hoog contrast, **hoog contrast licht** en **hoog contrast donker**. U kunt uw thema wijzigen door in het menu Help > themes te selecteren.

![Submenu Thema's][1]

De instelling van het thema wijzigt de kleur van de meeste elementen van de gebruikers interface. Het is raadzaam om ook het bijbehorende thema met hoog contrast in te scha kelen om ervoor te zorgen dat alle elementen van de gebruikers interface juist zijn gekleurd.
