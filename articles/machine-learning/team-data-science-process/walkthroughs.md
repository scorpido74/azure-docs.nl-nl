# <a name="walkthroughs-executing-the-team-data-science-process"></a>Stapsgewijze instructies voor het uitvoeren van het team data Science process

In deze **uitgebreide instructies** worden de stappen in het team data Science-proces voor specifieke scenario's gedemonstreerd. Ze illustreren het combi neren van Cloud, on-premises hulpprogram ma's en services in een werk stroom voor een **intelligente toepassing**. De instructies worden gegroepeerd op het **platform** dat ze gebruiken.


## <a name="walkthrough-descriptions"></a>Beschrijvingen van scenario's

Hier vindt u een korte beschrijving van de voor beelden van deze Walkthroughs op hun respectieve platformen:


- [HDInsight Spark-scenario's met PySpark en scala](walkthroughs-spark.md) Deze scenario's maken gebruik van PySpark en scala op een Azure Spark-cluster om predictive analytics uit te voeren.
- [HDInsight Hadoop-scenario's met hive](walkthroughs-hdinsight-hadoop.md) Deze scenario's maken gebruik van Hive met een HDInsight Hadoop-cluster om predictive analytics uit te voeren.
- [Azure data Lake-instructies met u-SQL](walkthroughs-azure-data-lake.md) Deze scenario's maken gebruik van U-SQL met Azure Data Lake om predictive analytics uit te voeren.
- [SQL Server](walkthroughs-sql-server.md) Deze scenario's maken gebruik van SQL Server, SQL Server R Services en SQL Server python-Services predictive analytics.
- [Azure Synapse Analytics](walkthroughs-sql-data-warehouse.md) Deze scenario's maken gebruik van Azure Synapse Analytics om predictive analytics uit te voeren.
