# <a name="solutions-and-training-for-azure-active-directory-b2c"></a>Oplossingen en trainingen voor Azure Active Directory B2C

Met Azure Active Directory B2C (Azure AD B2C) kunnen organisaties zakelijke oplossingen implementeren waarmee ze verbinding kunnen maken met hun klanten. De volgende oplossingen en trainingen zijn Download bare documenten waarmee u deze oplossingen kunt door lopen.
