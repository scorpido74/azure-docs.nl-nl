# <a name="service-bus-messaging-samples"></a>Voor beelden van Service Bus-berichten
In de Service Bus Messa ging-voor beelden worden de belangrijkste functies van [Service Bus Messa ging](https://azure.microsoft.com/services/service-bus/)gedemonstreerd. Op dit moment kunt u de voor beelden vinden op de volgende locaties:

| Programmeertaal | Locatie van de SDK of voor beelden |
| -------------------- | ----------------------- |
| .NET, Java en beheer | https://github.com/Azure/azure-service-bus/ |
| Node.js | https://github.com/Azure/azure-sdk-for-js/tree/master/sdk/servicebus/service-bus/samples |
| Python | https://github.com/Azure/azure-sdk-for-python/tree/master/sdk/servicebus/azure-servicebus |
| Aan de slag| https://github.com/Azure/azure-service-bus-go/ |

## <a name="service-bus-explorer"></a>Service Bus Explorer

Daarnaast is de [Service Bus Explorer](https://github.com/paolosalvatori/ServiceBusExplorer) een voor beeld dat wordt gehost op github, waarmee u verbinding kunt maken met een service bus service naam ruimte en eenvoudig bericht entiteiten kunt beheren. Het hulp programma biedt geavanceerde functies, zoals de functionaliteit voor importeren/exporteren, en de mogelijkheid om Messa ging-entiteiten en relay-services te testen. U vindt de volledige Service Bus Explorer-bron en-documentatie op [github](https://github.com/paolosalvatori/ServiceBusExplorer).
