# <a name="setting-up-access-for-azure-dev-tools"></a>Toegang instellen voor Azure Dev Tools

Er zijn twee manieren om toegang te krijgen tot uw abonnement, zodat u software kunt implementeren naar uw studenten en uw labs kunt inrichten:
1. Door software en sleutels te downloaden uit de Visual Studio Subscription-portal.
1. Door studenten toegang te geven tot de Education Hub Store.

Ga voor meer informatie naar de [website van Azure for Education](https://azureforeducation.microsoft.com).

## <a name="for-administrators-only"></a>Alleen voor beheerders  
Beheerders hebben toegang tot Azure Dev Tools via de [Visual Studio Subscription-portal](https://my.visualstudio.com/).

Als u toegang tot uw Visual Studio Subscription wilt aanvragen, kunt u dit doen via de [beheerportal](https://azureforeducation.microsoft.com/account/Subscriptions). Zodra u bent goedgekeurd, hebt u toegang tot de Visual Studio Subscription-portal. Als dit de eerste keer is dat u de Visual Studio Subscription-portal opent, wordt u gevraagd om een Microsoft-account te koppelen aan het e-mailadres dat is opgegeven tijdens de registratie. Om deze accounts te koppelen, hebt u de abonnements-id of het Benefits Access Number nodig, de voor- en achternaam van een beheerder, en het e-mailadres dat is opgegeven tijdens de registratie. Nadat u de accounts hebt gekoppeld, kunt u toegang krijgen tot de Visual Studio Subscriber-portal.

## <a name="for-students-faculty-and-administrators"></a>Voor studenten, faculteitsleden en beheerders
Studenten hebben toegang tot Azure Dev Tools via de [Education Hub](https://aka.ms/devtoolsforteaching).

Studenten en faculteitsleden hebben toegang tot alle voordelen van softwaredownloads via de Education Hub. De Education Hub is onderdeel van de Azure-portal en biedt uw studenten eenvoudig toegang tot de volledige softwarecatalogus, evenals toegang tot de complete [Microsoft Learn](/learn/)-catalogus.
