# <a name="manage-the-azure-maps-visual-within-your-organization"></a>De Azure Maps Visual in uw organisatie beheren

Power BI biedt ontwerpers en Tenant beheerders de mogelijkheid om het gebruik van de Azure Maps-Visual te beheren. Hieronder vindt u de stappen die elke rol kan uitvoeren.

### <a name="designer-options"></a>Opties voor ontwerpers

In Power bi Desktop kunnen ontwerpers de Azure Maps-Visual uitschakelen op het tabblad Beveiliging. Selecteer **Bestands** &gt; **Opties en instellingen** en selecteer vervolgens **Opties** &gt; **Preview-functies**. Als deze instelling is uitgeschakeld, wordt Azure Maps standaard niet geladen.  
  
![Power BI deel venster Opties waarin de opties voor preview-functies voor het visuele element van Azure Maps worden weer gegeven](media/power-bi-visual/preview-options-panel.png)

## <a name="tenant-admin-options"></a>Opties voor tenantbeheer

In PowerBI.com kunnen Tenant beheerders het Azure Maps visuele element uitschakelen voor alle gebruikers. Selecteer **instellingen** voor de &gt; **beheer** **Portal** &gt; **Tenant instellingen**. Als deze is uitgeschakeld, wordt de Azure Maps Visual niet meer weer gegeven in het deel venster visualisaties van Power BI.

> [!div class="mx-imgBorder"]
> ![Power BI beheer Portal met Tenant instellingen voor de Azure Maps Visual](media/power-bi-visual/tenant-admin-settings.png)
