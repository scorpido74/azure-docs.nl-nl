# <a name="azure-cli-samples-for-event-grid"></a>Azure CLI-voorbeelden voor Event Grid

De volgende tabel bevat koppelingen naar Azure CLI-voorbeelden voor Event Grid.

## <a name="event-grid-subscriptions"></a>Event Grid-abonnementen

- [Abonneren op Azure-abonnement](scripts/event-grid-cli-azure-subscription.md) - Hiermee abonneert u zich op gebeurtenissen voor een Azure-abonnement. 
- [Abonneren op Blob Storage](scripts/event-grid-cli-blob.md) - Hiermee abonneert u zich op gebeurtenissen voor een Blob Storage-account. 
- [Abonneren op aangepast onderwerp](scripts/event-grid-cli-subscribe-custom-topic.md) - Hiermee abonneert u zich op gebeurtenissen voor een aangepast onderwerp. 
- [Abonneren op resourcegroep](scripts/event-grid-cli-resource-group.md) - Hiermee abonneert u zich op gebeurtenissen voor een resourcegroep. 
- [Abonneren op resourcegroep en filter voor een resource](scripts/event-grid-cli-resource-group-filter.md) - Hiermee abonneert u zich op gebeurtenissen voor een resourcegroep en filtert u gebeurtenissen voor een resource. 

## <a name="event-grid-topics"></a>Event Grid-onderwerpen

- [Aangepast onderwerp maken](scripts/event-grid-cli-create-custom-topic.md) - Hiermee wordt een aangepast Event Grid-onderwerp gemaakt en worden het eindpunt en de sleutel geretourneerd.
